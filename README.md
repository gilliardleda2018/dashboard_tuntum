DASHBOARD ELEITORAL TUNTUM 2024

Arquivos esperados na mesma pasta do script:
- SEÇÕES.csv
- VOTACAO_PREFEITO.csv
- VOTAÇÃO_VEREADOR.csv

Como rodar:
1) Instale dependências:
   pip install streamlit pandas numpy plotly

2) Entre na pasta onde está o arquivo dashboard_tuntum_2024.py

3) Execute:
   streamlit run dashboard_tuntum_2024.py

O painel possui 3 visões:
- Prefeito 2024
- Vereadores 2024
- Exploração Territorial

Objetivo:
Identificar bairros fortes, competitivos e com maior potencial de crescimento eleitoral.
